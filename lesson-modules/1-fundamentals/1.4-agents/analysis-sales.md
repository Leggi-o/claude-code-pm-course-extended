## Patterns in Deal Losses

1. **Field/Mobile Workforce = Dealbreaker** (80% of lost deals)
2. **Mobile Web ≠ Native App** (prospects rejected mobile web as inadequate)
3. **Offline Mode is Table Stakes** (construction, field services demand it)
4. **Push Notifications Required** (enterprise expectations)
5. **Fastest to Act Wins** (Asana/Monday.com moved quickly)
