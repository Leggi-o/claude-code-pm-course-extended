# Claude Code for Product Managers - Course Materials

Welcome to the interactive Claude Code course for Product Managers!

## Getting Started

**You're in the right place!** This folder contains everything you need to complete the course.

### What's Included

- **`lesson-modules/`** - Interactive modules taught by Claude (Modules 0-2)
- **`company-context/`** - TaskFlow company reference materials for exercises
- **`.claude/`** - Slash commands that power the course experience

### How to Start the Course

**1. Make sure you're in this directory:**
```bash
pwd
# Should show: .../course-materials
```

**2. Launch Claude Code:**
```bash
claude
```

**3. Start the first module:**

Once Claude Code is running, type:
```
/start-1-1
```

This kicks off Module 1.1 - Welcome to Claude Code!

### Course Structure

**Module 0: Getting Started** (Prerequisites)
- Before running this course, you should have completed Module 0 (installation and setup)
- If you haven't installed Claude Code yet, visit: https://ccforpms.com/getting-started/installation

**Module 1: Fundamentals** (Starts with `/start-1-1`)
- 1.1: Welcome
- 1.2: Visualizing Files
- 1.3: First Tasks
- 1.4: Agents (Parallel Processing)
- 1.5: Custom Sub-Agents
- 1.6: Project Memory
- 1.7: Navigation

**Module 2: Advanced PM Work**
- 2.1: Write a PRD
- 2.2: Analyze Data
- 2.3: Product Strategy

### Slash Commands

Throughout the course, you'll use slash commands like:
- `/start-1-1` - Start Module 1.1
- `/start-2-1` - Start Module 2.1
- etc.

These commands are defined in the `.claude/` folder and guide you through each module interactively.

This course teaches you Claude Code **by using Claude Code**. You'll work with real files, real company context (TaskFlow), and real PM workflows. Claude is your guide, but you're driving the learning.

Ready to start? Type `/start-1-1` in Claude Code!

### What's Been Added

This extended version includes a **Quick reference guides/** folder with three comprehensive reference documents created during course completion:

1. **Level-1-Reference-Guide.md** - Complete reference covering Modules 1.1-1.3:
  - Course overview and TaskFlow context
  - Visual workspace setup techniques
  - File operations and @ reference syntax
  - Communication style templates
  - Image analysis best practices (including the critical Ctrl+V tip!)
  - Common PM workflows and patterns
  - Time savings summary

2. **Solutions-Research.md** - Design pattern solutions for TaskFlow user pain points:
  - 5 comprehensive design patterns (Empty State Design, Template Gallery, Digest Mode Notifications, Progressive Disclosure Onboarding, Sample Data Pre-population)
  - Real-world examples from Asana, Notion, Linear, GitHub, Stripe
  - Implementation details and effort estimates
  - Technical considerations
  - Competitive analysis
  - Success metrics
  - All sources from web search research

3. **research-communications.md** - Example of multi-format communication:
  - Slack Update format (casual, team-focused)
  - Executive Email format (strategic, business-impact)
  - Notion Document format (comprehensive, detailed)
  - Demonstrates communication style transformation using user research synthesis

These reference guides serve as quick-access resources for common Claude Code patterns and PM workflows learned throughout the course. They complement the original interactive teaching modules by providing standalone documentation you can reference anytime.
